# BooksCollector Automation Tests
Тестирования сервиса управления коллекцией книг.
## Что использовалась 
* Python 3
* Pytest
## Что было протестировано (Чек-лист)
В рамках проекта покрыты следующие сценарии:
1. **Добавление книг:**
   - Позитивные проверки (граничные значения 1 и 40 символов).
   - Негативные проверки (название более 40 символов).
2. **Работа с жанрами:**
   - Установка существующего жанра из списка.
   - Попытка установки несуществующего жанра (обработка ошибок).
   - Поиск книг по конкретному жанру.
3. **Возрастной рейтинг:**
   - Фильтрация книг, не подходящих детям (Ужасы, Детективы).
4. **Избранное:**
   - Добавление книги в избранное.
   - Защита от дубликатов в списке избранного.
   - Удаление книги из избранного.
## Запуск тестов
Для запуска команда:
**pytest -v tests.py**
